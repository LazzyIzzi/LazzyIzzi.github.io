LazzyIzzi's Projects

This site is currently under construction. 11/16/21

  • Java Libraries

    • MuMassCalculatorLib - A NIST-XCOM derived Java library for photon cross-section and other X-ray calculations
    • DistanceLib - 2D and 3D Euclidean and geodesic transforms and single pressure simulation of mercury injection capillary pressure

  • ImageJ plugins (MuMassCalculatorLib examples)

    • X-ray Calculator - Look-up and/or plot photon cross-section and atomic data
    • Beam Hardening Estimator - An interactive tool for optimizing radiographic and tomographic image acquisition.
    • CT Scan Simulator - Create mono or polychromatic X-ray attenuation sinograms by reprojecting 2D images 

  • ImageJ plugins (DistanceLib examples)

    • ExactEuclidean - Exact Euclidean distance map of 2D and 3D segmented images using Danielsson's algorithm.
    • Geodesic - Geodesic distance transform and tortuosity of 2D and 3D segmented images using distance propagation
    • HybridFloodFill_Scan - Geometry based simulation of mercury injection capillary pressure(MICP).
    • HybridFloodFill_Search - Bisection Search to find "Breakthrough" in MICP.

  • Excel VBA Projects

    • MuMassCalculator.xlam - An Excel Add-in for NIST-XCOM derived photon cross-sections and other X-ray calculations
    • CarShowHelper - An Excel application for managing car shows.
