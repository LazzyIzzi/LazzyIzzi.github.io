X-ray Calculator

Requires ImageJ 1.53k or higher and MuMassCalculatorLib.jar

Home

 

This plugin provides a non-modal dialog that allows the user to call up X-ray attenuation data for atoms and compounds while working with other images.

The X-ray Calculator plugin is built upon the Mu Mass Calculator library.

This figure shows the user dialog(A) and the three windows that are created or updated by X-Ray Calculator.

The top portion of the dialog is for obtaining the absorption cross sections in cm2/gm for a specific photon energy. Enter an atomic abbreviation for a single atom or enter a formula as atom1:count1:atom2:count2... as required by MuMassCalculator. Enter a photon energy in MeV in the dialog field and click the Calculate button. The cross sections will be listed in the QuickCalc window(B).

The bottom portion of the dialog is for plotting attenuation spectra from 1KeV to 100GeV or anywhere in between for the formula entered at (A) above. The user can select the energy range, the cross-sections to plot, and log or linear axes. Optionally, the absorption edge energies for the elements in the formula can be listed(C). Clicking the Update Plot button will create/update the windows C and D. The plot window(D) is ImageJ native with its set of built-in tools for expanding, listing and fitting the attenuation data.

Note: The results and plot window do not necessarily appear in the locations shown in the figure.
