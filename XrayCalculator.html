X-ray Calculator

Requires ImageJ 1.53k or higher and CT_Tools_Lib.jar

Home

This plugin provides a non-modal dialog that allows the user to call up X-ray attenuation data for atoms and compounds while working with other images.

This figure shows the user dialog and the three windows that are created or updated by X-Ray Calculator.

The top portion of the dialog is for obtaining the absorption cross sections in cm2/gm for a specific photon energy. Select a material from the "Material Name" pull down menu or, alternatively enter:

  • an atomic abbreviation for a single atom or enter a formula as atom1:count1:atom2:count2...
  • a density of the material.
Enter a photon energy in KeV in the dialog field and click the Calculate button. The cross sections will be listed in the QuickCalc window.

The bottom portion of the dialog is for plotting attenuation spectra from 1KeV to 100GeV or anywhere in between for the formula entered above. The user can select the energy range, the cross-sections to plot, and log or linear axes. Optionally, the absorption edge energies for the elements in the formula can be listed. Clicking the Update Plot button will create/update the results. The plot window is ImageJ native with built-in tools for expanding, listing and fitting the attenuation data.
