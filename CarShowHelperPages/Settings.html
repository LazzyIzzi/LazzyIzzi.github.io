The Settings dialog configures Car Show Helper for all steps of your show.
The Settings dialog shows only required fields based on your selections.

  1. Registration -
    • Car Numbers -
      • Use Old - Use for shows where cars are assigned permanent numbers. Imports old car numbers saved in the Contacts worksheet.
      • Use New - Automatically creates the next higher car number for each car at registration. You may also enter a car number manually.
    • First Car Number, begin new car numbers with this value (reserve lower car numbers for other uses).
    • Class Table - Select a Class table for Registration. User-defined Class tables are stored in the hidden Classes worksheet. See Edit Class Table to modify and save your own car classification tables. The Class Table is ignored for popular vote shows.
    • Use Contacts Sheet Classes - Check this box to import a car's class saved in the Contacts worksheet.
  2. Judging - Car Show Helper supports several commonly used car show types.
    • Popular Vote - Built-in, enter the votes per ballot in the popup box.
    • Popular Vote By Class - Built-in, Requires a Class table selection. Participants can vote for one car in each class.
    • Judging Tables - User-defined Judging tables stored in the hidden Judging worksheet. Select add or deduct points.
  3. Tally - Tally supports several judging methods
    • Favorites - Show winners are the ones with the most points. Enter the how many will get trophies in the popup. (The figure uses a judging sheet so the Favorites option is hidden.)
    • Most Points - Use with Popular Vote By Class. Show winners are the ones with the most points in each class.
    • Points Ranges - Use with shows where cars compete against fixed standards and not each other. You can change the ranges setting if a Tally produces an inappropriate number of trophy winners. Click Guess Pts Ranges to get a rough guess for points ranges. Some refinement will probably be needed based on your judging standards.
  4. Trophies -
    • Use 1st, 2nd, 3rd
    • Use First, Second, Third
    • Use Gold, Silver, Bronze - Recommended for Points Ranges shows.
    • Show All Cars - Tally normally hides cars that do not win a trophy. Check this box to show the points and awards for all registered cars in the Tally results.
