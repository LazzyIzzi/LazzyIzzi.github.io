Car Show Helper supports several commonly used car show types. The Settings dialog configures Car Show Helper for each step of your show. You can even change the configuration mid-show!

  1. Registration -
    • Car Numbers -
      • Use Old - Use for shows where cars are assigned permanent numbers. Imports old car numbers saved in the Contacts worksheet.
      • Use New - Automatically create a new car number for each car at registration. PopUp First Car Number, begin new car numbers with this value (reserve lower car numbers for other uses).
      • Enter Manually - Manually create a new car number for each car at registration.
    • Class Table - User-defined Class tables are stored in the hidden Classes worksheet. Select a Class table for Registration. The Class Table is ignored for popular vote shows.
    • Use Pre-assigned Classes - Check this box to import a car's class saved in the Contacts worksheet.
  2. Judging -
    • Popular Vote - Built-in, enter the votes per ballot in the popup box.
    • Popular Vote By Class - Built-in, Requires a Class table selection. Participants can vote for one car in each class.
    • Judging Tables - lists user-defined Judging tables stored in the hidden Judging worksheet. Select add or deduct points.
  3. Tally -
    • Favorites - Show winners are the ones with the most points. Enter the how many will get trophies in the popup.
    • Most Points - Use with Popular Vote By Class. Show winners are the ones with the most points in each class.
    • Points Ranges - Use with shows where cars compete against fixed standards and not each other
