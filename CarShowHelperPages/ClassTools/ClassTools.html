Edit Class Table

Back To Top

Click the Show Help button for instructions on how to use this dialog. Changes to the active class table are NOT made in the registration table.

Class Tables are located in the hidden "Classes" sheet. Use Data Tools->Show Data Sheets to show the "Classes" sheet. You can copy, edit and name class tables using the usual Excel methods. The Green columns are for user inputs. The actual class name in the Yellow column is generated using the "ClassMaker" formula . Car Show Helper will automatically add tables with the ".ClassTab" extension to Class Table selection menus.

The examples all use A,B,C etc. as main and sub class names but this is not a requirement. You can name Main and Sub Classes that work for your show. Car Show Helper uses Excel sorting and filtering methods to create the tally reports. Select main and subclass labels with Excel sorting in mind.

Use Data Tools->Hide Data Sheets to hide the "Classes" sheet.

Combine Sub Classes

Combines two selected sub-classes in the current Class Table into a new class. Updates all instances of the old classes in the "Registration" sheet with the new class. Classes are combined two-at-a-time. Combine multiple classes by multiple calls to Combine Sub Classes

Complete the pink fields then click OK

  • Main Class - A pull down menu of the main classes in the current class table
  • Sub Class To Combine 1 - A pull down list of the sub classes in the current main class
  • Sub Class To Combine 2 - A pull down list of the sub classes in the current main class. Select a different class than Sub Class 1
  • New Sub Class - A sub class name for the combined class consistent with the naming convention for the current main class
  • Start Year - the beginning year for the new combined class
  • End Year - the ending year for the new combined class
  • Description - The types of vehicles in the new combined class

In the example, all instances of class AA-Pre-1942-1942---Car Stock and AB-1943-1960---Car Stock will be changed to AA-Pre-1942-1960---Car Stock in the "Registration" table and the current class table

Split Class

Splits a class in the current Class Table into two new classes. Updates all instances of the old class in the "Registration" sheet with the new classes. Classes are split two-at-a-time. Split a class into multiple classes by multiple calls to Split Class

  1. Select the class you want to split
  2. Select a split criterion.
  3. Click "Get New Class Name Suggestions".
  4. Edit the suggestions if needed.
  5. Click "OK".
  • Class To Split - A pull down list of the classes in the current Class Table
  • Split Criterion - a class can be split by Year, Make or Model. Note "<" and ">" ignored for Makes and Models.
    • split criterion.
  • New Main Class - A main class consistent with the naming convention for the current class table
  • New Sub Class - A sub class consistent with the naming convention for the current class table
  • New Start Year - the beginning year for the new class
  • New End Year - the ending year for the new class
  • Description - The types of vehicles in the class
