Create Class Table

Jump To Class Tools Menu

Class Tables are located in the hidden "Classes" sheet. Create Class Table unhides the "Classes" sheet and shows the dialog. Enter a name for the new class table and click OK. A new bare class table template will be created in the next available column.

The Green columns are for user inputs. The actual class name is generated by a formula in the Yellow column. The Pink column is for information purposes and contains a count of cars currently registered in the class. Do not edit the Yellow or Pink columns.

Alternatively, you may copy an existing table like the one in the figure, rename it ending with "ClassTab" and edit the Green cells. Car Show Helper will automatically adds tables ending with "ClassTab" to the Class Table menus

Car Show Helper uses Excel sorting and filtering methods to create the show winner tally reports. Select main and subclass labels with Excel sorting in mind. The examples all use A,B,C etc. to begin the class name

The class tables columns are:
  • Main Class - a generic label for the class. Creates a pull-down menu divider if Sub Class is blank
  • Sub Class - a generic label for the sub class.
  • Start Year - the beginning model year for the sub class
  • End Year - the ending model year for the sub class
  • Description - a generic description of the vehicles in the class, e.g. Car, Truck, Stock, Modified, Ford, Ferrari etc.
  • Classes - Contains a formula that creates a class descriptor from the Green cell data.
  • Car Count - shows the number of cars currently registered in the class.

Use Data Tools->Hide Data Sheets to hide the "Classes" sheet.

Combine Classes

Combines two selected classes in the current Class Table into a new class. Updates all instances of the old classes in the "Registration" sheet with the new class. Classes are combined two-at-a-time. Combine multiple classes by multiple calls to Combine Classes

  • Class To Combine 1 - A pull down list of the classes in the current Class Table
  • Class To Combine 2 - A pull down list of the classes in the current Class Table. Select a different class than Class 1
  • Main Class - A main class consistent with the naming convention for the current class
  • Sub Class - A sub class consistent with the naming convention for the current class
  • Start Year - the beginning year for the new class
  • End Year - the ending year for the new class
  • Description - The types of vehicles in the class

In the example, all instances of class AA-1920-1929---Cars and AB-1940-1949---All will be changed to AA-1920-1949---Ford in the "Registration" table and the current class table

Split Class

Splits a class in the current Class Table into two new classes. Updates all instances of the old class in the "Registration" sheet with the new classes. Classes are split two-at-a-time. Split a class into multiple classes by multiple calls to Split Class

  • Class To Split - A pull down list of the classes in the current Class Table
  • Split Criterion - a class can be split by Year, Make or Model. Note "<" and ">" ignored for Makes and Models.
  • Get Suggestions - Generates suggested class names based on the split criterion.
  • Main Class - A main class consistent with the naming convention for the current class table
  • Sub Class - A sub class consistent with the naming convention for the current class table
  • Start Year - the beginning year for the new class
  • End Year - the ending year for the new class
  • Description - The types of vehicles in the class
