HybridFloodScan

Requires DistanceMapLib.jar

Home

This plugin performs a 3D simulation of the non-wetting imbibition process in a "porosity" image by calling HybridFlood at equal increments of pore radius between a selected range. In a porosity image, open(resolved) voxels = 1, closed(solid) voxels = 0 and everything in between 0 and 1 is unresolved porosity. See the HybridFloodFill page for more details about the HybridFlood process.

Like HybridFlood, HybridFloodScan operates only on 32-Bit stacks and supports anisotropic voxel dimensions. Check the options as needed. "Save Images" writes the image from each step of the scan, and "Show Plot"(recommended) displays a plot of flooded volume vs pore radius. Click "OK" to start the search.

HybridFloodScan produces similar outputs the the Search plugin but it does not identify breakthrough. It is intended to provide an digital estimate of the pore(throat) size distribution similar to mercury injection capillary pressure measurements. If "Show Plot" is selected the volume vs radius plot is displayed.

If "Save Images" was selected, the flooded images are written to tif files and can be used for visualization or other purposes.
